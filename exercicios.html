Exercícios

Exercício 1
Este código analiza o numero digitado e revela se ele é positivo ou negativo.
Exercício 2
Este código orienta sua faixa etária de acordo com a idade digitada.
Exercício 3
Este código relaciona os dias da semanas com números e de acordo com o número digitado revela qual dia é.
Exercício 4
Este código imprime no corpo da página de forma ordenada, números de 0 á 10.
Exercício 5
Este código soma todos os valores digitados e apenas quando o número "0" é inserido o resultado é revelado.
Exercício 6
Este código seleciona um número aleatório de 0 a 10 e o usuário deve tentar advinhar que número é esse.
Exercício 7
Este código pede 5 números ao usuário e em seguida revela o maior e o menor número digitado.
Exercício 8
Este código calcula a média resultante dos números digitados pelo usuário.
Exercício 9
Este código revela todos os números maiores que o número digitado pelo usuário (vai de 0 á 10).
Exercício 10
Este código conta quantas vezes a palavra escolhida se repete em um baralho de palavras.
